# to-do-app
Essa página contém uma to-do list com alguns critérios, sendo eles:
1 - É necessário incluir uma tarefa no campo de texto. Caso tente prosseguir com o campo vazio, o sistema pedirá para que você insira uma tarefa.
2 - Cada tarefa terá sua prioridade no momento de sua criação, podendo ser atribuída de 3 formas diferentes:
- Alta
- Média
- Baixa
3 - À medida em que a tarefa muda sua prioridade, a cor também altera.
- Para prioridade alta, a cor é vermelha
- Para prioridade média, a cor é amarela
- Para prioridade baixa, a cor é verde
4 - Há a possibilidade de excluir uma tarefa existente (funcionalidade reconhecida através do ícone de "X"), porém há um double check para garantir que você não está excluindo incorretamente.
5 - Assim como o botão de excluir, também há um botão de conclusão de tarefa após ser finalizada (funcionalidade reconhecida através do ícone da marca "nike").
  
6 - O sistema não irá te impedir, caso você queira selecionar a prioridade antes de descrever a tarefa, será possível desde que não deixe o campo vazio.
  
7 - Após alterar uma tarefa para concluída, é possível retirar essa alteração clicando em cima do mesmo ícone em que clicou para concluir.
  
8 - Depois da confirmação de exclusão de uma tarefa, não há como resgatá-la.
  
9 - A funcionalidade de filtro possibilita a busca por data, prioridade e/ou palavra-chave.
